### 1. Overview
Automated stock market data extraction from CafeF and implemented Ichimoku technical analysis models.

### 2. Actions & Detailed Implementation
- **Data Extraction**: Built a Python script using BeautifulSoup and request parsing to automatically extract financial and stock market data from CafeF, replacing manual data gathering.
- **Technical Analysis Modeling**: Integrated Ichimoku Cloud technical analysis algorithms to evaluate market trends and generate automated buy/sell signals based on real-time price feeds.

### 3. Key Results
- **Quantitative**: Automated data collection processed historical price charts for multiple tickers in seconds, eliminating manual data entry time.
- **Qualitative**: Ensured system stability and data reliability, providing automated evaluations based on real-time market data.
